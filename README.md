# 🌙 어드민나잇

시은 & 지수의 어드민나잇 주 2회 목표 트래커

## 기능
- 📅 **주간 캘린더** — 가능한 날 투표 + 완료 체크
- 🗓 **월간 캘린더** — 한눈에 완료 현황 + 개인 목표
- 📝 **할 일** — 시은/지수 각자 이번 주 할 일 관리
- 🔄 **실시간 공유** — 둘 다 같은 앱 쓰면 자동 동기화

---

## 🚀 Vercel 배포 방법

### 1. GitHub에 올리기
1. github.com 접속 → **New repository**
2. 이름: `adminnite`, Public 설정 → **Create**
3. 이 폴더 파일들 전부 업로드 (또는 git push)

### 2. Vercel 연결
1. vercel.com 접속 → **Add New Project**
2. GitHub 연동 → `adminnite` 레포 선택
3. Framework: **Vite** 자동 감지됨
4. **Deploy** 클릭

### 3. 완료!
- Vercel이 URL 발급 (예: `adminnite.vercel.app`)
- 시은이랑 지수 둘 다 그 URL 북마크하면 끝!
